# Entrega-N-3
Realicé una prueba estadística KS comparando células mieloides versus sanas, implementé un ajuste de curvas para un modelo biológico con múltiples parámetros, y analicé la dinámica del sistema tumor-inmune usando un modelo Lotka-Volterra extendido con datos de transcriptómica,  estudiando los mecanismos de resistencia a inmunoterapia anti-PD-1.
